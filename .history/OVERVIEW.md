# 📊 Overview
>github_basic

Ce document fournit une vue d’ensemble du projet github_basic, son objectif, sa structure, ses versions et son évolution.

---

### 🎯 Objectif du projet
Créer un projet pédagogique permettant d’apprendre :

- Git (commits, branches, merges, rebase)
- GitHub (issues, PR, milestones, releases)
- Workflows CI/CD
- Organisation professionnelle d’un dépôt

---
