# 📝 github_basic
Ce document contient les notions essentielles pour apprendre Git.

---
### 📌 1. Initialiser un dépôt
```git
git init
```

### 📌 2. Vérifier l’état du dépôt
```git
git status
```

### 📌 3. Ajouter des fichiers au suivi
```git
git add
```

### 📌 4. Créer un commit
```git
git commit -m "Message clair et descriptif"
```

### 📌 5. Ajouter un dépôt distant
```git
git remote add origin https://github.com/Teremu/githubbasic.git
>> (github.com in Bing)
```

### 📌 6. Envoyer le code
```git
git push -u origin main
```

###,📌 7. Créer une branche
```git
git checkout -b feature/ma-fonction
```

### 📌 8. Fusionner une branche
```git
git merge feature/ma-fonction
```

>Ces notes servent de mémo rapide pour progresser efficacement avec Git.


# 📝 Bases
## 📌 Commandes essentielles
```md
- `git init` — Initialise un dépôt
- `git status` — Affiche l’état du dépôt
- `git add .` — Ajoute les fichiers au staging
- `git commit -m "message"` — Crée un commit
- `git log` — Historique des commits
- `git branch` — Liste les branches
- `git checkout -b feature` — Crée et bascule sur une branche
```

### 📦 Objectif
>Ce fichier sert de mémo pour les commandes Git de base.
