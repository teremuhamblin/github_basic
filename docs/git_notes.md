📝 Notes Git — github_basic

Ce document contient les notions essentielles pour apprendre Git.

---

📌 1. Initialiser un dépôt
`
git init
`

📌 2. Vérifier l’état du dépôt
`
git status
`

📌 3. Ajouter des fichiers au suivi
`
git add .
`

📌 4. Créer un commit
`
git commit -m "Message clair et descriptif"
`

📌 5. Ajouter un dépôt distant
`
git remote add origin https://github.com/Teremu/githubbasic.git (github.com in Bing)
`

📌 6. Envoyer le code
`
git push -u origin main
`

📌 7. Créer une branche
`
git checkout -b feature/ma-fonction
`

📌 8. Fusionner une branche
`
git merge feature/ma-fonction
`

---

Ces notes servent de mémo rapide pour progresser efficacement avec Git.
