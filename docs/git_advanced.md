# 🚀 Git
>Concepts Avancés

### 🔀 Branches & Workflow
```md
- `main` : branche principale
- `feature/*` : développement de nouvelles fonctionnalités
- `hotfix/*` : corrections urgentes
```

### 🧩 Rebase

- `git rebase main`
- Permet de réécrire l’historique proprement

### 🧪 Tests & CI
>Intégration continue via GitHub Actions pour automatiser :

- Lint
- Tests
- Vérifications de structure

### 🔐 Sécurité

- Ne jamais commit un `.env`
- Utiliser `.gitignore`
