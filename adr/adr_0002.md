# 📁 ADR/ 0002
- Choix du workflow CI

### 🎯 Décision
Utiliser GitHub Actions comme système d’intégration continue.

### 📌 Contexte
Le projet doit illustrer les workflows GitHub.

### 🧩 Choix retenu
- Workflow simple : exécution du script principal
- Ajout futur : lint, tests, release automatique

### ✔️ Justification
- Intégré à GitHub
- Simple à configurer
- Parfait pour un projet pédagogique

### 🔮 Alternatives rejetées
- GitLab CI
- Jenkins
