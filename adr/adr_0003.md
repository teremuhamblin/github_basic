# ADR 0003
- Mise en place des tests unitaires

### 🎯 Décision
Utiliser pytest pour les tests unitaires.

### 📌 Contexte
À partir de la v0.3.0, le projet doit intégrer des tests.

### 🧩 Choix retenu
- Dossier tests/
- Tests simples pour commencer
- Intégration dans GitHub Actions

### ✔️ Justification
- Simple à apprendre
- Très utilisé dans l’industrie
- Compatible avec CI/CD

### 🔮 Alternatives rejetées
- unittest (trop verbeux)
- nose2 (moins populaire)
