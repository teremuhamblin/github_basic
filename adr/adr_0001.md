# 📁 ADR/ 0001
- Structure du projet

### 🎯 Décision
Adopter une structure simple et pédagogique pour le projet github_basic.

### 📌 Contexte
Le projet sert de support d’apprentissage Git/GitHub.  
La structure doit être claire, lisible et évolutive.

### 🧩 Choix retenu
```rst
src/
docs/
.github/
history/
```

### ✔️ Justification
- Facile à comprendre pour les débutants
- Compatible avec les workflows GitHub
- Évolutif pour les futures versions

### 🔮 Alternatives rejetées
- Structure monolithique
- Arborescence trop complexe
